# Free-Hairdrier
A Simple try in Cryptography
##意外发现
*并不确保生成的验证码必定生效*
---
据本人在曾居住过的两个楼层实现发现均满足一定规律；
---
吹风机的六位验证码满足这样的一个特定规律：
*d4 ≡ d1+d2+d3+d5+d6+offset (mod 10)*
---
此外呢，据知情人士爆料，它还满足这样一个规律：
*在曾经付费生成的验证码的第四位和第六位分别+1（不进位），得到的验证码也是可以使用的*
---
而据本人观察，六位验证码的第二位和第三位又常常与固定数字相关联：
*比如本人曾多次使用该吹风机，发现在本人居住的楼层其第二位常常为5，而第三位常常为1，且算出offset = 3*
---
因而诞生了本程序

##在不同楼层的使用指南
建议先观察一下曾使用过的验证码代码
---
**计算出第二位、第三位、以及offset的值**
在提供的freehairdrier.py程序的开始第五行修改设定offset的值即可奔放使用

##tk窗口指南
运行程序后会产生tk窗口
---
可设定第二位第三位的值
---
选择检验、生成、等价代换或批量生成均可奔放使用
